04.02.01 Javascript controlled animation

Here we go again with our character!
In this project, we had to build a website with our character and various buttons.
By pressing on the buttons the character will change his attributes such as opacity, positions, color, etc...
The real challenge was making sure that the user knew which button was pressed and which action was running.
We basically had to highlight the pressed button, but if the user pressed another button while another one was already pressed and the action was running the new button will have to be highlighted and the previous one which was highlighted will have to return normal.

04.04.01 Onboarding

The Onboarding assignment was one of the most difficult ones for us.
We were divided into groups of 4/5 and we had to come up with an interactive web-based onboarding sequence that could explain an important aspect of moving to Denmark through some games.
We, of course, started with a brainstorm, everybody in the group talked about their ideas.
Then we wanted to divide the project into tasks and assign each task to a member of the group, but unfortunately, nobody was THAT good at coding so we all had to help a little bit in coding while doing our main task.
I don't really like coding but it's good training every time I have to do it.
I was more focused on graphics such as the main character or the background images.
Our "game" was really simple compared to the others but we really liked how it turned out because everybody gave 110% for that project! The teachers liked it too!
